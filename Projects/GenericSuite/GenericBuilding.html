Generic Building

Simple building system with no hassle in Unreal Engine

What is it?

Is is a plugin that allows for users to place and remove placeable actors in the world with ease. It's based around the building systems seen in games like Rust, Valheim, or even ARK.

It supports valid and invalid placement with an interface as well as grid snapping.

How to use it

Generic Building is very easy to use, simply add the component to your character and set up its properties. To place a blueprint in the world call the StartPlacement() function from the component and pass in the GBuildData.
that is composed of the blueprint class you want to place and a static mesh for the actor.

It also provides an interface that can be implemented by the actors that you want to be placeable, this interface will allow you to customize the behavior of the building system for each actor.

Examples

In the example below I showcase the code for placement verification. This is all sill WIP.

CanPlace code snippet:

bool UGBuilderComponent::CanPlace_Implementation()
{
	UStaticMesh* Mesh = CurrentItemData.ProjectionMesh.LoadSynchronous();
	FTransform Transform = PlacementMeshComponent->GetComponentTransform();

	if (!Mesh) return false;

	// Local-space bounds (center + extents)
	const FBoxSphereBounds LocalBounds = Mesh->GetBounds();
	const FVector LocalExtents = LocalBounds.BoxExtent;
	const FVector LocalCenter  = LocalBounds.Origin;

	// Scale extents by world scale (rotation is handled by box rotation below)
	const FVector WorldScale = Transform.GetScale3D();
	const FVector ScaledExtents = LocalExtents * WorldScale.GetAbs();

	// World-space center of the box
	const FVector WorldCenter = Transform.TransformPosition(LocalCenter);
	const FQuat   WorldRot    = Transform.GetRotation();

	FCollisionQueryParams Params(SCENE_QUERY_STAT(CanPlace), false);

	if (APlayerController* PC = Cast(GetOwner()))
	{
		Params.AddIgnoredActor(PC->GetPawn());
	}
	else
	{
		Params.AddIgnoredActor(GetOwner());
	}

	FCollisionShape Shape = FCollisionShape::MakeBox(ScaledExtents);

	if (bDebug)
	{
		DrawDebugBox(GetWorld(), WorldCenter, ScaledExtents, WorldRot, FColor::Red, false, 1.f, 0, 2.5f);
	}

	FCollisionObjectQueryParams ObjectParams;
	ObjectParams.AddObjectTypesToQuery(ECC_WorldDynamic);
	ObjectParams.AddObjectTypesToQuery(ECC_Pawn);
	ObjectParams.AddObjectTypesToQuery(ECC_PhysicsBody); // optional

	const bool bBlocked = GetWorld()->OverlapAnyTestByObjectType(
		WorldCenter,
		WorldRot,
		ObjectParams,
		Shape,
		Params
	);
	
	return !bBlocked && PlacementHitResult.bBlockingHit;
}
